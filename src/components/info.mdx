<Heading>Info</Heading>

This app allows you to make Google Calendar events more quickly by parsing a simple shorthand notation that looks like this:

<Code type="pre">
    t, d, 2d, 3w buy some bananas
</Code>

This will create the same event today (<Code>t</Code>), in one day (<Code>d</Code>), in two days (<Code>2d</Code>) and in three weeks (<Code>3w</Code>). But it won't create the event yet, it will add it to the review area.

You can now press the enter key if you want to add additional events.

In the review area, you can choose to delete or send events. Just note that it will send to the calendar defined in the environment file, not your personal calendar (for security reasons).

You can also re-add past events to your review area by clicking on them.

Click the *info* button on the top-right at any time to re-open this dialogue.
